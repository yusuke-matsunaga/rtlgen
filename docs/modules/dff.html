class rtlgen.dff.Dff(parent, *, name=None, data_in=None, data_type=<class 'rtlgen.data_type.BitType'>, clock, clock_pol='positive', reset=None, reset_pol=None, reset_val=None, enable=None, enable_pol=None)

ベースクラス: ClockedProcess

D-FF を表すクラス

パラメータ:

  • name (str) -- 名前

  • data_in (Expr) -- データ入力

  • data_type (DataType) -- データ入力の型

  • clock (Expr) -- クロック入力

  • clock_edge (str) -- クロックのアクティブエッジを表す文字列 'positive' か 'negative'

  • reset (Expr) -- リセット入力

  • reset_pol (str) -- リセットの極性を表す文字列 'positive' か 'negative'

  • enable (Expr) -- イネーブル入力

  • enable_pol (str) -- イネーブル信号の極性を表す文字列 'positive' か 'negative'

以下の入出力を持つ． * data_in: データ入力．タイプは任意 * clock: クロック入力．タイプは BitType * reset: リセット入力(オプショナル)．タイプは BitType * enable: イネーブル入力(オプショナル)．タイプはBitType * q: 出力．タイプは data_in と同じ．
